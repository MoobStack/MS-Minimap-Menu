MS Minimap Menu legacy migration bridge
=======================================

This temporary addon loads OctoMinimapMenuDB so MS Minimap Menu 1.0.12 can
copy the former launcher position, appearance settings, exclusions, and custom
button names into MSMinimapMenuDB. It does not load the old minimap-menu
implementation. After migration has been verified and the game has saved the
new database during logout, this folder can be disabled or removed.
