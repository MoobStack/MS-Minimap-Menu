MS Minimap Menu 1.0.12
========================

This release prevents world-map POIs and other full world-map assets from being collected as minimap launcher buttons. The legitimate stock minimap World Map button remains available.

Install both sibling folders directly under Interface\AddOns:

  MSMinimapMenu
  !MSMinimapMenuCapture

After updating from 1.0.11, run:

  /msminimap scan
  /msminimap status

No saved-variable migration is required.
